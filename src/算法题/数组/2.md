## 思路

双指针-拷贝覆盖

[图解](https://leetcode.cn/problems/remove-duplicates-from-sorted-array/solutions/683841/shuang-zhi-zhen-shan-chu-you-xu-shu-zu-z-3pi4/)

快指针：遍历整个数组
慢指针：新数组的下标

```js
var removeElement = function (nums, val) {
  let k = 0;
  for (let i = 0; i < nums.length; i++) {
    if (nums[i] != val) {
      //   nums[k] = nums[i];
      //   k++;
      // 等价于 k++是先赋值再++
      nums[k++] = nums[i];
    }
  }
  return k;
};
```
