## 题目链接

[反转字符串](https://leetcode.cn/problems/reverse-string/)

## 思路

双指针交换顺序

## 实现

```js
var reverseString = function (s) {
  let len = s.length;
  let l = 0,
    r = len - 1;
  while (l < r) {
    [s[l], s[r]] = [s[r], s[l]];
    l++;
    r--;
  }
  return s;
};
```
