## 题目链接

[https://leetcode.cn/problems/maximum-depth-of-binary-tree/description/](二叉树的最大深度)

## 思路

1. 递归：对于当前节点，其最大深度 = 左右子树的最大深度+1
2. 层序遍历：看结果的长度

## 实现

```js
// 递归遍历
var maxDepth = function (root) {
  if (!root) {
    // 从0开始
    return 0;
  }
  return Math.max(maxDepth(root.left), maxDepth(root.right)) + 1;
};
```
