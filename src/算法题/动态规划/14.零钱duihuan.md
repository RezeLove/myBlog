## 题目链接

[零钱兑换][https://leetcode.cn/problems/coin-change-ii/description/]

## 思路

1. 完全背包
2. 求方法数：dp[0]初始化为 1

## 实现

```js
var change = function (amount, coins) {
  let dp = Array(amount + 1).fill(0);
  dp[0] = 1;
  for (let i = 0; i < coins.length; i++) {
    for (let j = coins[i]; j <= amount; j++) {
      dp[j] += dp[j - coins[i]];
    }
  }
  return dp[amount];
};
```
